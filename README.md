# Websitelerim

Bu repo, farklı websitelerimi düzenli bir şekilde saklamak için kullanılır.

## Klasör Yapısı

```
Test/
├── README.md
├── LICENSE
└── websites/
    ├── _template/          # Yeni site için şablon
    │   ├── index.html
    │   ├── css/
    │   │   └── style.css
    │   └── js/
    │       └── main.js
    ├── site-adi-1/         # İlk website
    │   ├── index.html
    │   ├── css/
    │   ├── js/
    │   └── img/
    └── site-adi-2/         # İkinci website
        ├── index.html
        └── ...
```

## Nasıl Kullanılır?

### Yeni bir website eklemek
1. `websites/_template` klasörünü kopyalayın
2. Yeni klasöre sitenizin adını verin
3. Dosyaları düzenleyin

### Belirli bir website üzerinde çalışmak
Claude'a hangi site üzerinde çalışmak istediğinizi söyleyin:
- *"websites/blog sitesinde çalışalım"*
- *"portfolio sitesine yeni bir sayfa ekle"*
